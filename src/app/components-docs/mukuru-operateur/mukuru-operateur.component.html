MUKURU OPERATEUR

Pour se connecter il faut suivre les étapes suivantes : Saisir le lien common sur la barre d’adresse du navigateur pour accéder à la page login

  • https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

Dans le menu Transfert d’argent, cliquer sur « Mukuru » et la page d’accueil apparaitra

Sur la page d'accueil Mukuru, outre l'accueil il y a deux onglets :

  • Retrait : permet d’effectuer les retraits
  • Reporting retrait : permet de retracer le rapport des retraits

RETRAIT | PAIEMENT

Pour faire un retrait :

  • Cliquez sur l’onglet « Retrait », puis un formulaire apparaitra de la manière suivante
  • Entrez le code retrait dans le champ ci-dessous et cliquez sur le bouton « rechercher »
  • Une fois que vous cliquez sur le bouton rechercher, le résultat de la recherche apparaitra
  • Veuillez remplir tous les champs correctement avec les informations du client :
    • givenName : Prénom du bénéficiaire
    • Surname : Nom du bénéficiaire
    • Genre : renseignée l’information qui apparait dans le résultat : male(masculin)
    • DateofBird : renseignée l’information qui apparait dans le résultat : 1999-12-20

NB: les éléments d’une transaction revienne dynamique en fonction des éléments renvoyer à l’envoie, il faudrait le remplir avec les informations corrects du bénéficiaire, dans le cas contraire l’opération ne sera pas about

VERIFICATION TRANSACTION

Pour vérifier une transaction :

  • Cliquez sur l’onglet « Reporting Retrait »
  • Et vous aurez l'interface suivante

Dans la page ci-dessus, vous pouvez filtrer les transactions par :

  • code retrait,
  • Periode (Date début & fin),
  • Devise ou,
  • Statut

NB: Si des transactions ont été effectuées pendant la période indiquée ou un des éléments mentionnés ci-haut, un tableau affichera les transactions correspondantes.
