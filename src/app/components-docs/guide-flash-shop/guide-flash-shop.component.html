Gestion Stock

1. GESTION STOCK

Pour se connecter il faut suivre les étapes suivantes :

  • Saisir le lien sur la barre d’adresse du navigateur et valider : https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider


Veuillez cliquer sur :

  • VAS


Ensuite cliquer sur :

  • Flash Stock


Une fois que l’utilisateur accède Flash Stock, la page suivante s’affiche :

A. VENTE

Pour effectuer une vente, l’opérateur procède de la manière suivante :

  • Entre le code du produit, puis faire une recherche (Avec la recherche avancée, il suffit d’un mot clé de la description, le model ou le nom du fabricant du produit).
  • Renseigne les numéros des séries


  • Une fois que la série estrenseignée,on ajoute le nom du client et son adresse puis on enregistre.


LISTE DES FACTURES

Pour voir les différentes factures passées sur la vente, voici les étapes à suivre :

  • Cliquer sur liste des factures
  • Filtrer (selon la période)


RAPPORT DES VENTES

Pour voir les différentes opérations passées sur la vente, voici les étapes à suivre :

  • Cliquer sur liste des factures
  • Filtrer (selon la période)


B. STOCK

ETAT STOCK

L’opérateur sait vérifier l’état de stock pour voir tous les mouvements de la manière suivante :

  • En renseignant le dépôt
  • et en filtrant par période


EXTRAIT DE SORTIE DE STOCK

Pour vérifier l’extrait de sortie de stock, L’opérateur procède de la manière suivante :

  • Renseigne le dépôt
  • Filtre selon la période


C. APPROVIONNEMENT

DEMANDE D’APPROVISIONNEMENT

Pour faire une demande d’approvisionnement l’opérateur procède de la manière suivante :

  • Mettre le code du produit,
  • Renseigne la quantité et ensuite enregistrer la demande


LISTE DES DEMANDES

Pour consulter la liste des demandes, l’opérateur procède de la manière suivante :

  • Renseigne le dépôt,
  • Filtre selon la période


D. LIVRAISON

ACCUSER DE RECEPTION

Pour accuser réception, l’opérateur doit avoir au préalable reçu un mail dans lequel il y’aura le numéro de bon de livraison (BL N°) et ensuite le renseigner sur le champ demandé.


LISTE DES BONS

Pour consulter la liste des bons, l’opérateur procède de la manière suivante :

  • Renseigne le dépôt,
  • Filtre selon la période


GENERER LE BON DE LIVRAISON

Après validation de la demande, il faudra générer un bon de livraison. Le bon de livraison sera générer en cliquant sur le bouton « Générer un BL ».


Il faut d’abord saisir le code du produit, ensuite renseigner le numéro de série de chaque produit à livrer et enfin donner le nom du livreur et son adresse (facultatif).
