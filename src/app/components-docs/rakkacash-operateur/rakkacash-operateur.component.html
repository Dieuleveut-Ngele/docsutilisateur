RAKKACASH OPERATEUR

Pour se connecter il faut suivre les étapes suivantes : Saisir le lien common sur la barre d’adresse du navigateur pour accéder à la page login

  • https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

Dans le menu Transfert d’argent, cliquer sur « Rakkacash » et la page d’accueil apparaitra

Sur la page d'accueil Rakkacash outre l'accueil il y a quatre onglets :

  • Cashin : permet d’effectuer l’envoi
  • Cashout : permet d’effectuer le retrait
  • Reporting Cashin : Affichage des rapports cashin
  • Reporting Cashout : Affichage des rapports cashout

CASHIN

Pour faire un cashin :

  • Cliquez sur l’onglet « cashin »,
  • Puis un formulaire apparaitra, remplissez les champs obligatoires(*),
  • Cochez la case « J'atteste avoir vérifié que toutes les données sont correctes »,
  • Et validez la transaction.
  • Une fois la transaction validée, il y aura un deuxième écrans DETAILS CASHIN afin d’avoir la confirmation de la transaction,
  • Validez la transaction en cliquant sur OUI
  • Une fois que vous avez valider la transaction vous aurez ceci sur votre écran « Reçu CASHIN »
  • Une fois que le reçu est affiché l’operateur peut imprimer en cliquant sur « Impression Thermique »

REPORTING CASHIN

Pour vérifier une transaction cashin :

  • Cliquez sur l’onglet « Reporting Cashin »
  • Dans le formulaire ci-dessus, précisez la période en renseignant la date Début et date Fin
  • Cliquer sur « Filtrer »

CASHOUT

Pour faire un CashOut :

  • Cliquez sur l’onglet « cashout », puis un formulaire apparaitra
  • Entrez le PIN dans le champ à gauche,
  • Préciser le montant au deuxième champ ensuite, cliquer sur le bouton « rechercher »

Le résultat de la recherche

  • Renseigner les informations obligatoires(*),
  • Cochez la case « J'atteste avoir vérifié que toutes les données sont correctes »,
  • Et validez la transaction.

REPORTING CASHOUT

Pour vérifier une transaction cashin :

  • Cliquez sur l’onglet « Reporting CashOut »
  • Dans le formulaire ci-dessus, précisez la période en renseignant la date Début et date Fin
  • Cliquer sur « Filtrer »
