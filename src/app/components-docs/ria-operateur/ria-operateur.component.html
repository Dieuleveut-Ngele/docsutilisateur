RIA OPERATEUR

Pour se connecter il faut suivre les étapes suivantes : Saisir le lien common sur la barre d’adresse du navigateur pour accéder à la page login

  • https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

Dans le menu Transfert d’argent, cliquer sur « Ria » et la page d’accueil apparaitra

Sur la page d'accueil Ria, outre l'accueil il y a deux onglets :

  • Retrait : permet d’effectuer les retraits
  • Transactions : permet de retracer le rapport des retraits

RETRAIT | PAIEMENT

Pour faire un retrait :

  • Cliquez sur l’onglet « Retrait », puis un formulaire apparaitra de la manière suivante
  • Entrez le PIN dans le champ à gauche et cliquez sur le bouton « Rechercher »,
  • Le résultat de la recherche apparaitra
  • Remplissez les champs obligatoires(*),
  • Cochez la case « J'atteste avoir vérifié que toutes les données sont correctes »,
  • Et validez la transaction. Une fois la transaction validée, il y aura un reçu qui apparaitra.

VERIFICATION TRANSACTION

Pour vérifier une transaction :

  • Cliquez sur l’onglet « Transactions »
  • Sélectionnez « retraits »
  • Dans le formulaire ci-dessous, précisez la période en renseignant sur la date Début et Fin
  • Cliquer sur « Filtrer »

Si des transactions ont été effectuées pendant la période indiquée, un tableau affichera les transactions correspondantes.
