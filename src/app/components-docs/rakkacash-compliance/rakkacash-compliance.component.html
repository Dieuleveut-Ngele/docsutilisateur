RAKKACASH COMPLIANCE

Pour se connecter il faut suivre les étapes suivantes : Saisir le lien common sur la barre d’adresse du navigateur pour accéder à la page login

  • https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

Dans le menu Transfert d’argent, cliquer sur « RAKKACASH » et la page d’accueil de RAKKACASH apparaitra

La page est composée de 4 menu :

  • Reporting cashin : Permet de visualiser la liste des cashin,
  • Reporting cashout : Permet de visualiser la liste des cashout,
  • Identité clients : Permet de visualiser les pièces ID des clients,
  • Blacklist : Permet de bloquer un client et de débloquer les clients blacklistés.

1. REPORTING CASHIN

Sur la page d’acceuil, cliquer sur le menu Reporting Cashin

Dans cette page, vous pouvez filtrer les transactions par :

  • Pays
  • Région
  • Ville
  • Agence
  • Code retrait
  • Date début & Date fin
  • Devise
  • Statut
  • Beneficiaire
  • Deposant
  • Numero
  • Numero Piece ID
  • ID Transaction

En cliquant sur le bouton Filtrer

Le reporting cashin filtrer s’affiche, vous pouvez exporter en Excel, en cliquant sur le bouton Exporter

4. REPORTING CASHOUT

Sur la page d’acceuil, cliquer sur le menu Reporting Cashout

Dans cette page, vous pouvez filtrer les transactions par :

  • Pays
  • Région
  • Ville
  • Agence
  • Code retrait
  • Date début & Date fin
  • Devise
  • Statut
  • Beneficiaire
  • Deposant
  • Numero
  • Numero Piece ID
  • ID Transaction

En cliquant sur le bouton Filtrer

Le reporting cashout filtrer s’affiche, vous pouvez exporter en Excel, en cliquant sur le bouton Exporter

3. IDENTITE CLIENTS

Sur la page d’acceuil, cliquer sur le menu Identité clients

Dans cette page, vous pouvez filtrer les transactions par :

  • Prenom
  • Nom
  • Postnom
  • Numero ID
  • Type de piece d’identite
  • Pin

En cliquant sur le bouton Filtrer

4. BLACKLIST

Sur la page d’acceuil, cliquer sur le menu Blacklist

Dans cette page vous pouvez :

  • Rechercher un client par son numéro ID et au moins un nom en cliquant sur le bouton Rechercher
  • Insérer dans la liste des clients blacklister
  • Exporter la liste des clients blacklistés en excel en cliquant sur le bouton exporter

NB: le bouton supprimer dans la colonne Action, permet de débloquer un client blacklisté
