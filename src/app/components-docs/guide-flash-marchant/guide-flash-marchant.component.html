Guide Flash Marchand

1. FLASH MARCHAND

Flash Marchand est un module de paiement des produit marchand sur Common Interface Pour y accéder l’operateur saisi le lien ci-dessous et se connecte

  • Saisir le lien sur la barre d’adresse du navigateur et valider : https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider


Veuillez cliquer sur :

  • VAS

Ensuite cliquer sur :

  • FLASH MARCHAND


2. PAIEMENT

Dans cette page l’operateur clique sur le menu Paiement



Dans la page suivante l’operateur doit saisir :

  • Le nom du client
  • Le prénom du client
  • Le numéro de téléphone


Apres avoir saisir l’identité du client, l’operateur sélectionne d’abord:

  • La devise du paiement
  • Puis le marchand
  • Ensuite l’Article à payer
  • Et Il saisi la quantité
  • Puis il clique sur le bouton ESTIMATION


Apres avoir cliquer sur Estimation, le résumeé de l’opération s’affiche :

  • Apres vérification, l’operateur coche la case (J’atteste avoir vérifier que toutes les données sont correctes)
  • Puis il clique sur le bouton VALIDER


Le reçu de l’opération s’affiche



Pour imprimer l’operateur clique sur le bouton IMPRIMER



Cliquer sur le bouton OK pour quitter



3. REPORTING

Pour y accéder cliquer sur le menu REPORTING

Dans cette page l’operateur peut :

  • Filtrer par devise
  • Il sélectionne la date début et date fin
  • Puis il clique sur le bouton Filtrer


Les differentes transactions s’affichent



l’operateur peut exporter les données en EXCEL en cliquant sur le bouton Exporter
