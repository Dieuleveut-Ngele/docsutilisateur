OPERATION MTN

Pour se connecter il faut suivre les étapes suivantes :

  • Saisir le lien sur la barre d’adresse du navigateur et valider : https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

Pour passer une opération MTN, l’opérateur clique sur

  • FLASH TRANSFERT

Une fois sur FLASH TRANSFERT, la page suivante s'affiche

1. ENVOI

Pour faire un envoi, cliquer sur Envoi
