Bulk Notification

1. BULK NOTIFICATION

Le Bulk Notification est une fonctionnalité d'envoyer des notifications à plusieurs Flash en même temps. Le format du fichier à uploader doit être du csv avec une seule colonne contenant les FP. Vous les trouverais dans :

  • VAS

Ensuite veuillez cliquer sur :

  • BULKSMS


Sur la page ci-dessus veuillez cliquer sur le menu :

  • Envoyer des notifications

Sur la page suivante :

  • Cliquer sur parcourir pour joindre le excel contenant la liste des FP forma CSV
  • Saisir le titre de notification
  • Saisir le texte de notification
  • Cliquer sur le bouton envoyer


NB: le statut Opération Echouer sera afficher si est seulement si, le pos n’a pas appliquer les mise à jour du nouveau apk flahsapp sur son mobile


2. REPORTING NOTIFICATION

Pour le reporting, cliquer sur le bouton Reporting Notification



Sélectionner les dates puis cliquer sur le bouton Appliquer le filtre
