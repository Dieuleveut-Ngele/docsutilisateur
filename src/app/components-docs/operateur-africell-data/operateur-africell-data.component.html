2. RECHARGE DATA

Pour procéder à l’opération de DATA AFRICELL, il faut cliquer sur le menu Recharge mega

Pour passer les opérations de recharge de mega AFRICELL, l’opérateur procède de la manière suivante :

  • Saisir le numéro Africell « au format : 09 »
  • Sélectionner la devise du payement « CDF, USD ou FCN »
  • Cliquer sur le bouton « Charger Forfaits »
  • Ensuite cliquer sur Menu pour selectionner la validité de votre forfaits « exemple 1 jour »
  • Ensuite cliquer sur FORFAIT pour sélectionner le forfait de votre choix « exemple : 1000 MB »
  • Une fois le forfait sélectionner le montant de la transaction sera affiché

Par la suite l’opérateur clique sur le bouton Valider, pour une dernière vérification de l’opération Une boite de dialogue est affichée avec les informations de la transaction :

  • Le numéro de téléphone
  • Le forfait choisi et le montant à payer

Pour finir, l’opérateur clique sur le bouton OUI et pour annuler l’opération, l’opérateur clique sur le bouton NON

Après avoir cliquer sur le bouton OUI, un SUCCESS vous est affiché pour signifier que votre opération a réussie

3. REPORTING

L’opérateur sait voir ses opérations DATA AFRICELL en cliquant sur « Transactions » puis sur « Recharges mega »

Après avoir cliqué sur le menu « Transaction » puis Recharges mega, la page suivante s’affiche

Voici les actions que l’opérateur sait exercer : en cliquant sur le bouton Filtrer

  • Filtrer l’opération par rapport à un POS ID
  • Filtrer l’opération par rapport à un N° de téléphone
  • Filtrer les opérations par rapport à une période donnée (date début et date fin)
  • Filtrer les opérations par rapport à la devise
  • Consulter le montant total des transactions filtrées (en CDF et USD)
  • Exporter la liste des opérations Data Africell sur un fichier Excel.
