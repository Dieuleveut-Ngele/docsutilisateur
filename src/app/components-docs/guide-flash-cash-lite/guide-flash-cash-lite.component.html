Guide Flash Cash Lite

OPERATION FLASH CASH LITE

Pour se connecter, l’operateur saisi le lien ci-dessous

  • https://commons.cfc-rdc.com:3000 sur la barre d’adresse du navigateur
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

Pour passer une opération Flash Cash Lite, l’opérateur clique sur

  • FLASH TRANSFERT

Une fois sur FLASH TRANSFERT, la page suivante s'affiche

1. ENVOI

Pour faire un envoi, cliquer sur :

  • Envoi
  • Selectionner le pays de destination
  • Suivant

Pour passer les opérations d’envois Flash Cash Lite, l’opérateur procède de la manière suivante :

  • Cliquer sur « Envoi »
  • Sélectionner « FlashCash Lite »

Après avoir cliqué sur « FlashCash Lite », la page suivante s’affiche :

N.B : l’opérateur est appelé à prendre connaissance de l’information repris ci-dessous :

Les comptes flashPOS ne sont pris en considération pour les retraits dans common interface, cette fonctionnalité est donc reservée aux agents CFC

Une fois sur la page d’envoi flash cash lite, l’opérateur procède de la manière suivante :

  • Le pays d’origine est déjà présélectionné ;
  • Sélectionner le pays de destination ;
  • Renseigner le montant d’envoi ;
  • Sélectionner la devise ;
  • Sélectionner « Accusé de réception » pour que l’expéditeur soit notifier par SMS du retrait de son bénéficiaire (le frais des SMS sont inclus lors de l’opération);
  • Cliquer sur le bouton « Estimation »

Après avoir estimé, le bloque suivant de l’estimation va s’afficher à coté avec informations telles que :

  • Le montant principal
  • Le frais
  • Le montant total
  • Le montant de retrait
  • Saisir le numéro de téléphone de l’expéditeur et du bénéficiaire (+243…) ;
  • Scanner la pièce d’identité du client ;
  • Cocher « j’ai bien vérifié toutes les données »
  • Cliquer sur le bouton « valider »

2. RETRAIT

Pour passer les opérations de retrait Flash Cash Lite, l’opérateur procède de la manière suivante :

  • Cliquer sur Retrait
  • Sélectionner Lite

Une fois que l’opérateur clique sur « Lite », la page suivante s’affiche :

N.B : l’opérateur est appelé à prendre connaissance de l’information repris ci-dessous :

Les comptes flashPOS ne sont pris en considération pour les retraits dans common interface, cette fonctionnalité est donc reservée aux agents CFC

Une fois sur la page de retrait flash cash lite, l’opérateur procède de la manière suivante :

  • Saisir le PCN ;
  • Saisir le numéro de téléphone (de l’expéditeur) ;
  • Cliquer sur le bouton « vérifier et envoyer code validation » et un code de validation est envoyé au numéro du bénéficiaire ;
  • Saisir le code de validation reçu par le bénéficiaire
  • Scanner la pièce identité du bénéficiaire ;
  • Cocher la case pour la confirmation des données saisies ;
  • Cliquer sur le bouton « valider » pour finaliser votre opération

Après validation de l’opération de retrait, la fenêtre de confirmation suivante s’affiche :

3. REPORTING

L’opérateur sait voir les différentes transactions d’envois et de retraits effectués on procédant de la manière suivante :

  • Cliquer sur Transaction,
  • Cliquer sur Lite

Après avoir cliqué sur « Lite », la page suivante s’affiche avec les possibilités suivantes :

  • Filtrer selon la période, la devise, le statut, le PCN
  • Cliquer sur filtrer
  • L'opérateur peut exporter les transactions en cliquant sur le bouton « Exporter »
