SABABALAR FINANCE

Pour se connecter il faut suivre les étapes suivantes : Saisir le lien common sur la barre d’adresse du navigateur pour accéder à la page login

  • https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

1. PARAMETRAGE TAUX

Pour paramétrer le taux du produit Sababalar, cliquer sur le bouton

  • Compta
  • Puis cliquer sur le bouton Taux par produit

Dans la page suivante :

  • Sélectionner le produit Sababalar
  • Sélectionner la devise Origine: Devise 1
  • Sélectionner la devise Destination : Devise 2
  • Saisir le montant
  • Saisir le montant (taux indicatif) : Optionnel

Puis cliquer sur le bouton Valider

2. PARAMETRAGE DE LIMITE

Pour paramétrer la limite Sababalar, cliquer sur le bouton

  • Compta,
  • puis cliquer sur le bouton Limite Transaction produit

Pour paramétrer la limite :

  • Sélectionner le Produit Sababalar
  • Sélectionner le type : IN (inbound) ou OUT (outbound)
  • Sélectionner le type de la période : transactionnelle, journalière ou mensuelle
  • Saisir le montant et la devise (commentaire en appui)
  • Cocher la case Blocage(pour bloquer la transaction lorsque la limite est atteint)
  • Cocher Envoi de mail en le renseignant dans le champs adresse e-mails(un mail sera envoyé automatiquement à l’adresse renseigné)

Puis cliquer sur le bouton valider

Un récapitulatif vous sera affiché, cliquer sur Oui pour confirmer et Non pour annuler

Si vous avez bien paramétrer vous aurez le tableau suivant

NB: pour modifier une limite existante, il faut refaire le paramétrage et celle-ci viendra écraser l’ancien paramètre (prise en compte immédiate)

3. COMMISSION PAR TRANCHE

Pour paramétrer la commission (Sag), aller dans menu

  • Compta
  • puis cliquer sur le bouton commission par tranche

Pour paramétrer la commission :

  • Sélectionner le Produit Sababalar
  • Sélectionner la catégorie de SAG
  • Sélectionner le type de transaction: Payement (Actuellement) et la devise des transactions
  • Sélectionner le type de commission : Sur le Principal ou le Frais
  • Préciser les plages (saisir le minimum et le maximum du montant principal)
  • Saisir le pourcentage commission (en devise) ou pourcentage commission FCN (en FCN) a recevoir sur le montant principal ou le frais
  • Mettre un commentaire

Puis cliquer sur le bouton valider

Un récapitulatif vous sera affiché, cliquer sur Oui pour confirmer et Non pour annuler

Si vous avez bien paramétrer vous aurez un tableau pareil

NB: Pour modifier un paramètre existant, il faut aller dans la colonne Action et cliquer sur le bouton modifier soit supprimer la ligne et recommencer le paramétrage et celle-ci prendra effet immédiatement.

4. REPORTING

Aller dans le menu :

  • VAS
  • Puis cliquer sur le bouton « SABABALAR » et la page d’accueil apparaitra
  • Sur la page d’accueil, cliquer sur l'onglet Reporting tickets

Dans le menu Reporting Tickets, Vous trouverez des champs a remplir :

  • Region
  • ville
  • TicketId
  • PosId
  • Telephone
  • Date début et Date fin
  • Devise
  • Statut

Puis cliquer sur le Bouton Filtrer et vous pouvez exporter en Excel, en cliquant sur le bouton Exporter
