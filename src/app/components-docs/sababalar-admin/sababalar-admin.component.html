SABABALAR ADMIN

Pour se connecter il faut suivre les étapes suivantes : Saisir le lien common sur la barre d’adresse du navigateur pour accéder à la page login

  • https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

AJOUT DU PRODUIT & HABILITATION CONFIG

1. AJOUT PRODUIT SABABALAR

  • Aller dans produits & config
  • Cliquer sur le bouton Produits
  • Saisir le « Sababalar » dans le champs Nom produit
  • Sélectionner type Service
  • Mettre l’état du « produit » en ON
  • Saisir un commentaire puis cliquer sur le bouton Valider
  • Puis confirmer en cliquant sur Oui
  • Et ensuite, se rassurer que le produit créer se retrouve dans la liste des produits

2. HABILITATION CONFIG

  • Aller dans Produit & Config, cliquer sur le bouton API CONFIG

Cliquer sur le bouton +Ajouter pour ajouter le nouveau paramètre

Renseigner les paramètres qui vous sera communiquer puis cliquer sur le bouton Valider

VISUALISER ET EXPORTER LES TRANSACTIONS

  • Aller dans le menu VAS, cliquer sur « SABABALAR »

La page d’accueil de SABABALAR apparaitra

REPORTING TCKETS

Sur la page d’acceuil, puis cliquer sur le menu Reporting Tickets

Dans ce tableaux vous pouvez filtrer les transactions par :

  • Pays
  • Ticket id
  • Région
  • Ville
  • Agence
  • PosId
  • Code retrait
  • Date début & Date fin
  • Devise
  • Statut

En cliquant sur le bouton Filtrer

Le reporting Tickets filtrer s’affiche, vous pouvez exporter en Excel, en cliquant sur le bouton Exporter
