MUKURU COMPLIANCE

Pour se connecter il faut suivre les étapes suivantes : Saisir le lien common sur la barre d’adresse du navigateur pour accéder à la page login

  • https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider

Dans le menu Transfert d’argent, cliquer sur « Mukuru » et la page d’accueil apparaitra

La page est composée de 4 onglets :

  • Reporting Retrait : Permet de visualiser la liste des transactions
  • Identité clients : Permet de visualiser les pièces ID des clients
  • Blacklist : Permet de bloquer un client et de débloquer les clients blacklistés
  • Limite transactionnelle : permet de paramétrer la limite transactionnelle des opérations Mukuru

1. REPORTING RETRAIT

Cliquez sur l’onglet « Reporting Retrait » et vous aurez un formulaire suivant

Dans cette page ci-dessus, vous pouvez filtrer les transactions par :

  • Pays
  • Région
  • Ville
  • Agence
  • Code retrait (Numéro de référence)
  • Par période (Date début et date fin)
  • Statut

En cliquant sur le bouton Filtrer

Vous pouvez également exporter les transactions en Excel, en cliquant sur le bouton Exporter

IDENTITE CLIENTS

Cliquez sur l’onglet « Identité clients » et vous aurez un formulaire suivant

Dans cette page ci-dessus, vous pouvez filtrer l’identité des clients par :

  • Prénom
  • Nom
  • Postnom
  • Agence
  • Date naissance et lieu
  • Numéro ID en sélectionnant le type de la pièce d’identité

En cliquant sur le bouton Filtrer

Vous pouvez également exporter les identités en Excel, en cliquant sur le bouton Exporter

BLACKLIST

Cliquez sur l’onglet « Blacklist » et vous aurez un formulaire suivant

Dans cette page ci-dessus vous pouvez :

  • Rechercher un client par son numéro ID et au moins un nom en cliquant sur le bouton « Rechercher »
  • Insérer dans la liste des clients blacklister
  • Exporter la liste des clients blacklistés en excel en cliquant sur le bouton exporter

En cliquant sur le bouton Filtrer

NB: le bouton supprimer dans la colonne Action, permet de débloquer un client blacklisté

LIMITE TRANSACTIONNELLE

Cliquez sur l’onglet « Limite Transactionnelle » et vous aurez un formulaire suivant

Pour paramétrer la limite transactionnelle (sur une transaction) il faut :

  • Sélectionner le type : ENVOI ou PAIEMENT "retrait"
  • Saisir le montant
  • Sélectionner la devise
  • Ecrire un commentaire

Puis cliquer sur le Bouton valider

Un récapitulatif sera affiché, cliquer sur Oui pour confirmer ou Non pour annuler

Vous pouvez également exporter les transactions en Excel, en cliquant sur le bouton Exporter

NB: Pour modifier une limite existante, il faut refaire le paramétrage et celle-ci viendra écraser l’ancien paramètre "prise en compte immédiate"
