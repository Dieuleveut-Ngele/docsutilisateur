Guide MoneyGram

MONEYGRAM

Pour se connecter il faut suivre les étapes suivantes :

  • Saisir le lien sur la barre d’adresse du navigateur et valider : https://commons.cfc-rdc.com:3000
  • Sur la page affichée, entrer son identifiant et son mot de passe, puis valider


Veuillez cliquer sur :

  • MONEYGRAM


Sur la page d'accueil MoneyGram outre l'accueil il y a trois onglets :

  • Envoi : permet d’effectuer les envois
  • Retrait : permet d’effectuer les retraits
  • Transactions : permet de retracer le rapport des envois et des retraits


1. ENVOI

Pour faire un envoi :

  • Cliquez sur l’onglet « Envoi », puis un formulaire apparaitra


Remplissez le formulaire en sélectionnant :

  • le pays de destination,
  • la devise de destination,
  • le montant à envoyé.
  • Cliquez sur « charger mode paiement »
  • Choisissez le « Mode de paiement » dans le menu déroulant pour afficher le frais et le taxe du transfert, pour savoir le montant total que l’expéditeur doit donner pour ce transfert.


  • Renseignez les informations obligatoires de l’expéditeur et du bénéficiaire,
  • Cochez la case « J'atteste avoir vérifié que toutes les données sont correctes »,
  • Et validez la transaction.


Le récapitulatif apparaîtra, puis cliquez sur « OUI ». Après confirmation de la transaction, un reçu apparaîtra.
